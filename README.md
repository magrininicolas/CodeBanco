# Como Executar

## Localmente

- Clone o repósitorio do git
- Insira o seguinte comando no terminal

```
gcc main.c ContaBancaria.c -o nome-programa -lm
```

- Rode
```
./nome-programa
```

- Se tudo estiver correto, o seu terminal deverá apresentar o seguinte
